Logic

Logic is the study of argumentation or the laws of correct reasoning, as it provides the framework for evaluating the validity of arguments and the truth of it's statements.

It's can be seperated into 2 difference catagories:

  • Natural logic

    Natural logic refers to the informal reasoning that occurs in everyday language and thought. It involves intuitive understanding and reasoning without the strict formalism of symbolic systems.

  • Symbolic logic

    Symbolic logic involves the use of abstract symbols and systems to represent logical expressions and arguments. It provides a rigorous framework for analyzing reasoning.

Argument form by a set of premises and a conclusion which may form a proposition, the abstraction of an expression can only determined either true or false if it's a truth-bearer. The relationship between arguments or expressions can be described using logical connective.
