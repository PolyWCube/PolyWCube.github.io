Function

The concept of the dependency of a quanity is vary on other quanity. It is the set of instruction in order to form a resultant set from the given set. In this case, the given set could be called the input set or domain of the function, while the resultant set called the output set or the codomain of the function through out the process called function evaluation.

Corresponding to the argument or variable of a element of domain into the function, that element would maped a value associated to the domain.

Denotation

Function: f, h, s, v, g,...

The value or codomain element y at variable, argument or domain element x: y = f(x)

The value or codomain element y with the set A: y = f(A)
